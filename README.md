# AI-FIREWALL

## Overview

AI-FIREWALL is a real-time network threat detection system that monitors network traffic and identifies malicious packets using machine learning. The project captures packets, extracts network features, and classifies traffic as normal or malicious.

## Features

- Real-time packet capture
- Network traffic monitoring
- Malicious packet detection
- Dataset-based machine learning prediction
- Threat logging
- Simple dashboard for monitoring

## Technologies

- Python
- Scapy
- PyShark
- Pandas
- NumPy
- Scikit-learn

## Project Workflow

Network Traffic
        ↓
Packet Capture
        ↓
Feature Extraction
        ↓
ML Model Prediction
        ↓
Threat Detection
        ↓
Alert & Logging
